# SecureATM-System

A console-based ATM Management System developed in C++ using Object-Oriented Programming concepts.

## Features

- Secure PIN authentication
- PIN hashing for basic security
- Deposit and withdrawal operations
- Account balance checking
- Mobile number update
- Transaction history tracking
- Account lock system after multiple failed attempts
- Multi-user account support
- Input validation handling

## Technologies Used

- C++
- OOP Concepts
- STL (Vector, String)
- File Handling Concepts
- Hashing using `std::hash`

## Project Structure

```bash
mini_ATM.cpp
```

## How to Run

### Compile

```bash
g++ mini_ATM.cpp -o atm
```

### Execute

```bash
./atm
```
